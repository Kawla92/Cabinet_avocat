⚖️ Landing Page – Cabinet Juridique

📌 Description

Cette landing page a été conçue pour un cabinet d’avocat dans le cadre d’un projet réel.
Le projet n’a finalement pas été retenu, mais il représente une étude complète en design et intégration web.

🎯 Objectifs

- Créer une interface claire et rassurante pour des clients juridiques
- Mettre en avant les services et la crédibilité du cabinet
- Optimiser la conversion (prise de contact)

🛠️ Technologies utilisées

- HTML5
- CSS3
- JavaScript
- Node.js (server.js)

📂 Structure du projet

- "index.html" → structure principale
- "styles.css" → design et mise en page
- "script.js" → interactions frontend
- "server.js" → logique backend simple
- "package.json" → dépendances

🚀 Lancer le projet en local

npm install
node server.js

Puis ouvrir :
http://localhost:3000

💡 Ce que j’ai appris

- Structurer une landing page orientée conversion
- Adapter un design à un domaine sensible (juridique)
- Gérer une séparation front/back simple

⚠️ Remarques

- Les données présentes sont fictives
- Aucune information réelle ou sensible n’est utilisée

📎 Statut

Projet non retenu par le client — utilisé ici comme démonstration de compétences
